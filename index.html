Tecnologías Implementadas

Machine Learning Espacial

Algoritmos de clustering espacial, redes neuronales convolucionales para análisis de imágenes satelitales, y modelos predictivos geo-referenciados.

Análisis Satelital

Procesamiento de datos LANDSAT, Sentinel y GPS de alta precisión para monitoreo ambiental y análisis de cambios temporales.

Redes Neuronales

Implementación de GNNs (Graph Neural Networks) para análisis de redes espaciales y detección de comunidades geográficas.
